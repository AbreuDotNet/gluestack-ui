# Text

Enhance your app with gluestack-ui's Text component—an adaptable React Native text area with multiple styles, sizes, and formatting options for seamless UI design.

This is an illustration of **Text** component.
/// {Example:basic} ///

<br />

## Installation

### Run the following command:
    ```bash
   npx gluestack-ui-nightly add text
    ```
<ManualInstallationNote />

## API Reference

To use this component in your project, include the following import statement in your file.

```ts
import { Text } from '@/components/ui/text';
```

```ts
export default () => <Text />;
```
