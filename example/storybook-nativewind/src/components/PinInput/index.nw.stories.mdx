## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import {   
  PinInput,
  PinInputField,
 } from '@/components/ui/pin-input';
```


```jsx
export default () => (
  <PinInput>
    <PinInputField />
  </PinInput>
);
```

#### PinInputField

It inherits all the properties of of react-native's [TextInput](https://reactnative.dev/docs/textinput) component on native and html input tag on web.
