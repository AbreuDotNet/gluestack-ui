## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import { useToast, Toast } from '@/components/ui/toast';
```

```jsx
export default () => (
  <Toast>
    <ToastTitle />
    <ToastDescription />
  </Toast>
);
```
