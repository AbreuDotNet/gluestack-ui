### Import

To use the component in your project, run the command below into your import statement. This will import the component from your components folder.

```bash
import { View } from '@gluestack-ui/themed';
```

### Props

gluestack-ui View component is created using View component from react-native. It extends all the props supported by [React Native View](https://reactnative.dev/docs/view#props), [utility props](https://ui.gluestack.io/docs/styling/utility-and-sx-props) and the props mentioned below.
