## Platform Specific

We can provide platform specific styles in the sx prop of the component. The sx prop takes an object with \_ios, \_andorid and \_web specific styles.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: `
      <Box justifyContent="center" alignItems="center" h="100%">
        <Box alignItems="center">
          <Button
            sx={{
              _ios: { bg: '$red500' },
              _web: { bg: '$green500' },
              _android: { bg: '$yellow500' },
            }}>
            <Button.Text>Button</Button.Text>
          </Button>
        </Box>
      </Box>
      `,
      transformCode: (code) => {
        return transformedCode(code);
      },
      scope: {
        Wrapper,
        Text,
        Button,
        Box,
      },
      argsType: {},
    }}
  />
</AppProvider>
