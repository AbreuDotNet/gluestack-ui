# Using Fonts

Fonts can be loaded into your project either by using Font plugin from [@dank-style/react]("https://dank.style/") or by configuring you app using google fonts, then directly using the font in gluestack config file.

## Font Plugin

[@dank-style/react]("https://dank.style/") exports `createStyled` and `FontResolver` functions which takes any Component as first parameter and applies the passed font with given font Weight, Size, Style to the component.

## First Step

For the font to work, we need to load that font in that app.

### For Next.js

To load font in Next.js

- Add import of font in global css file
- Link font using link tag in `<Head/>` in `_document.tsx` file
- Add font file locally in public folder and use @fontface in css file

### For Expo

To load font in Expo use useFonts function from [expo-font]("https://docs.expo.dev/guides/using-custom-fonts/#using-a-custom-font").

## Second Step

### Usage

To apply fonts to the Text Component we need to add config, this will return Text component with font styles added for both expo and Next.js .

Update the index.tsx file in `components/core/index.tsx` with below code:

```jsx
import { Root as Text } from './styled-components';
import { createStyled, FontResolver } from '@dank-style/react';

const styledFonts = createStyled([new FontResolver()]);

const StyledText = styledFonts(Text, {
  fontFamily: 'Smokum',
  fontWeight: 400,
  fontStyle: 'regular',
  fontSize: '$xl',
});

export { StyledText as Text };
```

Similarly, we can pass font styles to any component and then use the returned component (with font styles added).
