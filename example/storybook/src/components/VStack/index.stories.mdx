# Installation

This command copies the `VStack` component to your project.

```jsx
npx install @gluestack-ui add VStack
```

## Usage

```jsx
// import the component from your component folder
import { VStack } from 'components';

// Using the VStack component
export default () => (
  <VStack>
    <Box />
    <Box />
  </VStack>
);
```

# Advanced

## Customizing the VStack

We have a function called `createVStack` which can be used to create a custom VStack component. This function takes in a configuration object which contains the styled components that you want to use for the VStack. You can refer [dank.style](https://dank.style/) for more information on how to use styled components.

### Usage

```jsx
// import the styles
import { Root, Spacer } from './components/VStack';

// import the createVStack function
import { createVStack } from '@gluestack-ui/vstack';

// Understanding the API
const VStack = createVStack({
  Root,
  Spacer,
});

// Using the vstack component
export default () => <VStack />;
```

Default styling of the component can be found in the `components/VStack` file. For reference, you can view the [source code](https://github.com/gluestack/universa11y/blob/development/example/storybook/src/components/styled-components/vstack/index.tsx) of the styled `VStack` component.
