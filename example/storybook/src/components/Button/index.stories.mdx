# Installation

This command copies the `Button` component to your project.

```jsx
npx install @gluestack-ui add Button
```

## Usage

```jsx
// import the component from your component folder
import { Button } from 'components';

// Using the button component
export default () => (
  <Button>
    <Button.Text>Button</Button.Text>
  </Button>
);
```

## Anatomy

The button component consists of the following parts:

```jsx
export default () => (
  <Button.Group>
    <Button>
      <Button.Text>Button</Button.Text>
      <Button.Spinner />
    </Button>
  </Button.Group>
);
```

### Button.Spinner

Contains all spinner related layout style props and actions.
It inherits all the properties of React Native's [ActivityIndicator](https://reactnative.dev/docs/activityindicator#props) component.

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.
- Option to add your styles or use the default styles.

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the button's action.

### Screen Reader

- VoiceOver: When the button is focused, the screen reader will announce the button's label and its current state.

## Dependencies

- `@gluestack-ui/utils`
- `@react-native-aria/focus`
- `react-native-svg`

# Advanced

## Customizing the Button

We have a function called `createButton` which can be used to create a custom button component. This function takes in a configuration object which contains the styled components that you want to use for the button. You can refer [dank.style](https://dank.style/) for more information on how to use styled components.

### Usage

```jsx
// import the styles
import {
  Root,
  Text,
  Group,
  GroupHSpacer,
  GroupVSpacer,
  Spinner,
} from '../ui-components/Button/styled-components';

// import the createButton function
import { createButton } from '@gluestack-ui/button';

// Understanding the API
const Button = createButton({
  Root,
  Text,
  Group,
  GroupHSpacer,
  GroupVSpacer,
  Spinner,
});

// Using the button component
export default () => (
  <Button.Group>
    <Button>
      <Button.Text />
      <Button.Spinner />
    </Button>
  </Button.Group>
);
```

Default styling of all these components can be found in the `components/Button` file. For reference, you can view the [source code](https://github.com/gluestack/universa11y/blob/development/example/storybook/src/ui-components/Button/index.tsx) of the styled `Button` components.
